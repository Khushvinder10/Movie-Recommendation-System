# 🎬 Movie Recommendation System

## 📌 Project Overview

This project is a Movie Recommendation System developed using Python and Machine Learning techniques. It recommends movies similar to a movie entered by the user using TF-IDF Vectorization and Cosine Similarity.

## 🚀 Features

* Loads movie dataset from a CSV file.
* Performs data preprocessing.
* Converts movie information into numerical feature vectors using TF-IDF.
* Calculates similarity between movies using Cosine Similarity.
* Recommends the top similar movies based on user input.

## 🛠️ Technologies Used

* Python
* Jupyter Notebook
* NumPy
* Pandas
* Scikit-learn
* Difflib

## 📂 Dataset

* Dataset: `movies.csv`

## ▶️ How to Run

1. Open the notebook in Jupyter Notebook.
2. Place `movies.csv` in the same folder as the notebook.
3. Run all cells.
4. Enter a movie name when prompted.
5. View the recommended movies.
